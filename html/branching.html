Branching

Branching is the implementation of "if...else" statements when using JavaScript.

When running a function in JavaScript, you may want to have multiple possible results for a given action. For example, you might ask the user to choose a favorite color, and alter the background of the page depending on their answer. To accomplish this change, you could prompt the user to choose either red or green. If the user chooses red, the background is programmed to turn red, else if they choose green the background is programmed to turn green. You can add more else if conditions for other possible results. Else if they don't choose a color recognized by the page, you can tell them their choice was invalid as a fallback option.

$("#branching-button").click(function() {
    var color = prompt("Type either 'green' or 'red'!");
      if (color === "green") {
        $("body").removeClass();
        $("body").addClass("green-background")
      } else if (color === "red") {
        $("body").removeClass();
        $("body").addClass("red-background")
      } else {
        alert("You didn't choose a color! Try again!")
      }
  });

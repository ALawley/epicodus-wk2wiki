JavaScript Functions

A JavaScript function performs an action on an argument.

JavaScript functions perform actions on their arguments and deliver a result. For example, the prompt function presents a box for the user to enter information. The argument for that prompt function is what appear to (or prompt) the user. Other functions are alert, confirm, submit, or anything you can define using JavaScript (e.g. you can make your own functions!).

In the following example, we're calling an alert function after you press the button.

$("#functions-button").click(function() { alert("You clicked the button! Good job!") }; });

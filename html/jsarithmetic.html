JavaScript Arithmetic

JavaScript can be used to perform a number of arithmetical operations.

In JavaScript, we can add, subtract, multiply, and divide numbers by using +, -, *, and / respectively. We can also use exponents with ^, and modulo with %, which returns the remainder when two numbers are divided. We can also set numbers to variables and perform arithmetic upon them.

Box Model

The box model represents the space between text and the perimeter of an element, and between the perimeter of that element and the next. We can use it to style pages.

Within every element, padding buffers the space between the content being displayed within the element's borders, and margin buffers the space between the element's borders and the adjacent elements. We can define these spaces to change the element's layout, as well as create a visual representation of the border itself.

The following css will make the element with the #box-example id moved away from the previous element, and outline its border in red.

#box-example { margin-top: 50px; border: 2px solid red; }

Responsive design

Responsive design is the practice of making webpages function and display nicely on devices of various sizes (e.g. smartphones, tablets, computers, etc.).

There are multiple ways to make a site responsive. An easy way is to use Bootstrap and call their pre-defined size classes for your elements. Another way is to put media queries in your css file, telling the page to change the style at a breaking-point (when the screen is equal to a certain number of pixels). It's important not to set fixed-pixel-sizes for elements if you want a page to be responsive. Instead, you can use percentages to allow an element to re-size as the browser size changes.

The example below shows two images: the first is responsive, and will change size at different browser sizes. The second image has a fixed size and won't re-size.

#responsiveimg { width: 80%; } #fixedimg { width: 800px; }

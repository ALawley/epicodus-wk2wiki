Attributes

An attribute is a piece of information used to affect a html element.

Attributes help modify an element, such as a paragraph tag, to give it more specificity. For example, a class tag can be added and defined to style a specific paragraph. We can use jQuery to interactively modify attributes in the DOM without changing the page source.

In the following example we will show the JavaScript code that will change the page body background-color attribute when the button is clicked.

$("#attributes-button").click(function() {
    $("body").toggleClass("red-background")
  });

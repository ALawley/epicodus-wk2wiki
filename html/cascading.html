Cascading styles

When calling redundant items in css, it will read top-to-bottom and the last read item will overwrite conflicting items. This is called cascading and is very handy for styling.

When we have multiple paragraphs, we might have some styling we want applied to all paragraphs, but some styling may only apply to specific paragraphs. We can use cascading by setting the css for the specific paragraphs later, so that our css will style all paragraphs with a general rule, then overwrite the specific paragraphs with our cascaded style.

For example, here we will make all of our paragraph text larger and italic, and then cascade over that for paragraphs with the .specific class to make their font even larger. All of our text will be italic, with the .specific text being larger than the rest.

p {
  font-size: 1.2em;
  font-style: italic;
}

p .specific {
  font-size: 1.6em;
}

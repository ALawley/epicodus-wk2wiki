Div and Span

Div and span are html elements we can use to select certain parts of our page for specific styling.

We can wrap sections of our html in either the div tag or the span tag, letting us assign class and id tags to the selected sections for more granular styling. We can then use those classes and ids as selectors in css. Div is a block element, while span is an in-line element used for short snippets of html.

In the following example, we'll wrap a section of our page in a div tag with the class ".text-italic". A section within that will be wrapped in a span tag with the class ".text-big". Our css is set to style these classes accordingly.

By using divs, we can take advantage of classes to style different html elements with one single css rule!

  • It also lets us just style specific parts of text.
  • So our other lists and paragraphs won't get this styling!
  • Divs and spans are really handy!

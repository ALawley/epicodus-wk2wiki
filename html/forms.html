Forms

Forms allow you to get user input to manipulate with JavaScript to provide page interactivity.

Adding a form to your page allows users to input information, which you can then pass on in JavaScript in order to modify the page based upon user input. Forms can be easily added to your page with Bootstrap, however they won't actually do anything without JavaScript.

In the following example, we'll show you the JavaScript code to broadcast an alert back to the user with the string they enter into the form "#form-example", as well as the html to create the form (assuming you're using Bootstrap).

$("form").submit(function() { var formInput = $("input#message").val(); alert("You said '" + formInput + "'.") });
